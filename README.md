MCP Quant

MCP Quant is an AI-powered quantitative trading dashboard for Polymarket finance markets.

Features
Screens live Polymarket finance markets
Filters near-the-money opportunities
Calculates probabilities using:
EWMA volatility model
Historical simulation
Momentum scoring
Compares model probabilities with market probabilities
Calculates edge
Generates BUY YES, BUY NO, or PASS recommendations
Suggests position sizing
Maintains a trade journal for performance tracking
Technology
Python
Streamlit
Pandas
NumPy
SciPy
yfinance
Polymarket Gamma API
Roadmap
